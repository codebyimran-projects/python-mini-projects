

## Python Mini Projects

This repository contains multiple small Python projects. Each project is stored in its own folder and is fully implemented in a single file named `main.py`. These projects are created to practice Python fundamentals, logic building, and simple automation.

---

## Repository Structure

Each project follows the same simple layout:

```
project-folder/
    main.py
```

List of included projects:

* Qr-code-generator
* mini-cafe-management-system
* fractal-fern-generator
* rock-paper-scissors
* share-bill

New projects will continue to follow this same structure.

---

## Project Overview

### Qr-code-generator

A Python script that generates QR codes for any text or URL. The entire logic is written in `main.py`.

### mini-cafe-management-system

A basic console-based cafe management script. It performs menu display, order taking, and bill calculation inside a single `main.py`.

### fractal-fern-generator

Simple Python graphic demonstrations using built-in libraries. All drawing logic is in `main.py`.

### rock-paper-scissors

A classic Rock Paper Scissors game built with Python. The complete game logic is contained in `main.py`.

### share-bill

A bill-splitting script that calculates individual contributions based on total cost. Fully implemented in one file.

---

## How to Run Any Project

1. Open the required project folder.
2. Run the script using the command:

```
python main.py
```

Each project is self-contained and does not require additional files.

---

## Purpose

The aim of this repository is to keep Python projects simple, modular, and easy to understand. Each folder represents a complete working project with a single entry file.

---

## Notes

* No extra dependencies unless specified inside the project folder.
* All projects are written using Python 3.
